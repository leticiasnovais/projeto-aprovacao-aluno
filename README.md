# IA na prática: Acelerando o desenvolvimento e garantindo a qualidade com um fluxo de trabalho automatizado por IA

## Descrição do projeto

O objetivo do projeto foi demonstrar como ferramentas de Inteligência Artificial podem acelerar o desenvolvimento de software e melhorar sua qualidade utilizando o **GitHub Copilot** para auxiliar na escrita do código e dos testes automatizados, e o **GitHub Actions** para executar os testes automaticamente a cada alteração enviada ao repositório. Como exemplo, foi desenvolvido um programa simples em Python que verifica se um aluno foi aprovado ou reprovado com base na média de duas notas.

---

# Análise do problema

No cenário proposto, a equipe de desenvolvimento enfrenta dificuldades para equilibrar velocidade e qualidade. Quando prioriza entregas rápidas, a quantidade de testes diminui e aumentam os erros encontrados pelos usuários. Quando dedica mais tempo aos testes e revisões de código, os prazos deixam de ser cumpridos.

Os principais problemas identificados são:

- Desenvolvimento lento devido à escrita de código repetitivo;
- Baixa cobertura de testes automatizados;
- Descoberta tardia de erros durante os testes manuais ou em produção;
- Falta de padronização entre os desenvolvedores.

A proposta da atividade é demonstrar como a IA pode ajudar a resolver esse problema.

---

# O papel da Inteligência Artificial no desenvolvimento de software

A Inteligência Artificial vem sendo utilizada para automatizar tarefas repetitivas durante o desenvolvimento de software, o **GitHub Copilot** auxilia o desenvolvedor sugerindo códigos, funções, testes automatizados e boas práticas de programação, reduzindo o tempo gasto na implementação. Já o **GitHub Actions** automatiza processos de Integração Contínua (CI), executando testes automaticamente sempre que uma alteração é enviada ao repositório, dessa forma erros podem ser identificados rapidamente antes que o sistema seja disponibilizado aos usuários. A utilização conjunta dessas ferramentas melhora a produtividade da equipe e reduz falhas.
